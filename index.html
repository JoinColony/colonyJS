Colony SDK browser demos

See Colony SDK and colonyStarter for more information.

Check out the source code for the examples here.

These demos interact with LIVE Colonies on Gnosis chain. We recommend to use the Colony RPC endpoint directly.

  • Basic example (fetching funds from a Colony)
  • MetaMask example (connecting to MetaMask)
  • Advanced example (creating a domain within a Colony)
  • Events example (listening to Colony events)
  • Motion workflow (connecting to a Demo Colony)

These demos interact with a local development RPC endpoint (see here on how to set it up). MetaMask is not needed, the examples provide a wallet account.

  • Basic example for local setup
  • Reputation example (get reputation for a user)
  • Motion workflow
